Github Api Data Visualizations This service allows a user to search for a github repository and explore interactive visualizations of that repo's data. It works by targeting github's api servers then filterting and preparing the json to be presented and saved in the browser for later viewing without having to resend http requests to github. Subsequent searches utilize certain http headers like ETAG to optimize performance and reduce traffic.
